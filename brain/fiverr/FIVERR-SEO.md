## Title Formula (High-Converting)

**Current:** "n8n openclaw ai agent automation for voice and chat"

**Recommended Formula:**
```
[Main Service] + [Differentiator] + [Tech Stack] + [Use Case]
```

**Examples to Test:**
1. "AI Voice & Chat Agents | n8n Automation | OpenClaw | 24/7 Response"
2. "n8n AI Agents for Voice Calls, Chat, & Automation - OpenClaw Expert"
3. "Build AI Voice Agents & n8n Workflows | OpenClaw Automation Expert"

**Why This Works:**
- Keyword rich (voice, AI, n8n, automation, agent)
- Specific (not generic "automation")
- Uses power words (Build, Expert, 24/7)
- Includes differentiator (OpenClaw, Agent)

## Performance Thresholds & Benchmarks

### Healthy Metrics (Target)
- **Impressions:** 800+ per month (currently 500, down from 550)
- **CTR:** 3-5% (currently ~0.68%, needs improvement)
- **Orders:** 3-5 per month from organic
- **Response Time:** <2 hours (currently 2h, maintain)
- **Conversion Rate:** 1-2% (impressions → orders)

### Red Flags (Current)
- ⚠️ **Impressions dropping 40-50%** (Feb 5 → Feb 25)
- ⚠️ **CTR very low** (0.68% on peak day)
- ⚠️ **Orders not matching visibility** (500 impressions but sporadic orders)

### Action Triggers
- If impressions drop below 400/mo → Update thumbnail + title
- If CTR stays below 1% → Redesign thumbnail + test new tags
- If no orders 7+ days → Consider Promoted Gigs budget
